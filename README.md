# Banking_system_java
A beginner-friendly Java project that simulates basic banking operations using OOP concepts and user input handling.
# Banking System in Java

Features

- Create a bank account with name and initial balance  
- Deposit money  
- Withdraw money with validation  
- Check current balance  
- Display account holder information  
- Simple and user-friendly menu-driven interface  

Technologies Used

- Language:Java  
- Concepts Used: 
  - Object-Oriented Programming (OOP)  
  - Classes & Objects  
  - Constructors  
  - Encapsulation (private variables)  
  - Conditional Statements  
  - Loops  
  - Scanner for input
